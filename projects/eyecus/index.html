    视由 全称「视界事由」，本品牌规范包含了品牌的设计原则以及使用规则，通过对标志、色彩、图案、排版等方面的描述，保证品牌使用过程中的视觉统一。

    概述 Overview

    视界事由，英文名「eyecus」，简称「视由」，是一间来自中国的创意工作室，提供文化创意产业相关服务。

    Currently provide Cultural and Creative Industries(CACI) related services.

    标志 Logo

    标志构成 Logo Elements

    为了实现最大限度的简约与易读性，标志采用了最为基本的文字标志，由简体汉字视由与英文单词「eyecus」左右排列组成。

    其中汉字与英文字母两部分可视场景单独展示。标志只能从规范文件中直接拷贝，不能使用其他字体或重绘使用。

    标志系统 Logo System

    不同的使用场景应对标志排版进行相应调整。

    品牌色 Color

    品牌标准色 Basic Color

    「视由」品牌主色为 #3A5DAE，称之为 「视界蓝」，为了使标志颜色不过与单调，增加了一种辅助色#3CCBDA。

    视界蓝
    HEX: #3A5DAE
    RGB: (58,93,174)
    HEX:
     #3CCBDA
    RGB:
     (38,198,218)

    色彩系统 Color System

    品牌标准色颜色较深，适合用在一些较为正式的场合，在体现年轻活力方面表现力不足，于是确定了颜色更浅的次主色#008AD8与次辅色#00C19F来应对这些场合。

    色彩 应用 HEX值 RGB值 CMYK值 Pantone值
    		主色 #3A5DAE (58,93,174) (86,66,0,0) PMS 7455 C
    		辅色 #3CCBDA (60,203,218) (59,0,12,0) PMS 2226 C
    		次主色 #008AD8 (0,138,126) (78,33,0,0) PMS 2382 C
    		次辅色 #00C19F (0,193,159) (82,0,56,0) PMS 2240 C
