    Markdown 基础语法
    Markdown Web Grammar
    Published on 20190812, have 562 Words.

    Markdown 是一种轻量级标记语言，她轻量化、易读易写特性，并且对于图片，图表、数学式都有支持，语法也极其简单易学。

    Markdown 的创始人为约翰·格鲁伯 John Gruber。它允许使用易读易写的纯文本格式编写文档，然后转换成有效的HTML文档。这种语言吸收了很多在电子邮件中已有的纯文本标记的特性，目前许多网站都广泛使用 Markdown 来撰写说明文件或是用于论坛上发表讯息。本段文字摘自维基百科。

    标题

    输入

    ### 标题3 eyecus
#### 标题4 eyecus
##### 标题5 eyecus
###### 标题6 eyecus

    输出

    标题3 eyecus

    标题4 eyecus

    标题5 eyecus
    标题6 eyecus

    文本

    输入

    正常文本 eyecus 
**粗体 eyecus** 
*斜体 eyecus* 
~~删除线 eyecus~~
[链接 eyecus](https://eyecus.com)

    输出

    正常文本 eyecus

    粗体 eyecus

    斜体 eyecus

    删除线 eyecus

    链接 eyecus

    列表

    输入

    1. 正常文本1 eyecus

    2. 正常文本2 eyecus
