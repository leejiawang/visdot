    DNS，是英语 Domain Name System 的首字母缩写，中文名为域名系统。

    域名系统是互联网的一项服务。它作为将域名和 IP 地址相互映射的一个分布式数据库，能够使人更方便地访问互联网。DNS 使用 TCP 和 UDP 端口 53。当前，对于每一级域名长度的限制是 63 个字符，域名总长度则不能超过 253 个字符。 本段文字摘自 维基百科。

    修改 DNS 方法

    Windows 系统

    打开「控制面板」，依次点击「网络和Internet」、「以太网」、「属性」，双击 TCP/IPv4，选择自定义 DNS，修改并确定。

    使用快捷键「Win+R」 打开运行「cmd」，输入ipconfig /flushdns后回车以刷新下 DNS 缓存。

    macOS 系统

    打开「系统偏好设置」，依次点击「网络」、「高级」、「DNS」 选项卡，在左侧窗口中修改并确定。

    Linux 系统

    使用命令 vim /etc/resolv.conf，将首选「nameserver」修改并确定。
