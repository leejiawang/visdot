    使用 jsdelivr 免费加速你的网站
    Q&A Blog Web Html JavaScript Hexo Hugo Vuepress
    Published on 20201204, have 402 Words.

    使用「github pages」建立的博客虽然免费，但在国内加载速度实在太慢，使用国内服务器又被「备案」困扰，现在可以使用「jsdelivr」免费加速你的网站。

    CSS、JS、图片等资源均可用 Jsdelivr 进行加速，本文主要介绍通过其与 Github 配合加速图片资源，下面就看看要如何操作吧。

    配置 Github 仓库

    进入 Github 的官网，注册账户并登陆，建立一个仓库。将需要加速的图片、CSS、JS等资源上传至该仓库。

    给该仓库创建 「Releases」 版本，并记住自定义的版本号。

    调用 Jsdelivr

    使用如下命令调取资源，通过该链接访问的资源已被「Jsdelivr」进行加速，可直接在网页使用该链接调取资源。

    https://cdn.jsdelivr.net/gh/用户名/仓库名@版本号/资源名.后缀名

    假如 Github 用户名为 eyecus，仓库名为cdn，版本号为 1.0 ，资源所在的子文件夹为 img，资源为 hugo.svg，则资源加速链接为：

    https://cdn.jsdelivr.net/gh/eyecus/cdn@1.0/img/hugo.svg

    实际使用中也可不区分版本号，直接使用。

    https://cdn.jsdelivr.net/gh/eyecus/cdn@master/img/hugo.svg
