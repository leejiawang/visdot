    笔记「Html」
    Built in 20190705

    前端的作用于采集输入信息，后端进行处理。计算机程序的界面样式，视觉呈现属于前端。本页主要记录我学习 Html 过程中觉得实用的笔记。

    HTML 称为超文本标记语言，是一种标识性的语言。它包括一系列标签，通过这些标签可以将网络上的文档格式统一，使分散的资源连接为一个逻辑整体，下图为HTML基本框架示意图。

    <!DOCTYPE html>
<html>
  <head>
    <meta charset="utf-8">
    <title>eyecus</title>
  </head>
  <body>
    <h1>hello world！</h1>
    <p>hello <a>eyecus</a>！</p>
    <ul>
      <li>1.XXXX</li>
      <li>2.XXXX</li>
      <li>3.XXXX</li>
    </ul>
  </body>
</html>

    注意
    对于中文网页需要使用 <meta charset="utf-8"> 声明编码，否则会出现乱码。

    <!DOCTYPE html> ：声明为html5文档
    <html>：html页面的根元素
    <head>：包含了文档的元(meta)数据
    <title>：包含了页面标题
    <body>：包含了可见的页面内容
    <h1>：一级标题标签
    <p>：段落标签

    常见属性

    标签 描述
    class 为html元素定义一个或多个类名
    id 定义元素的唯一id
    style 规定元素的行内样式
    title 描述了元素的额外信息

    给元素增加链接
